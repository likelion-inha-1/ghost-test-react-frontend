# 👻 귀신 유형 테스트 — 애니메이션 & 디테일 기획

> 목적: 으스스한 분위기 조성 + 몰입 유지 · 작성일: 2026-07-19
> 관련 문서: [PLAN.md](../PLAN.md) · [DESIGN.md](./DESIGN.md)

## 0. 설계 원칙

1. **은은한 상시 + 강한 한 방** — 계속 움직이는 건 아주 미세하게, 강한 연출은 클라이맥스 한 곳에만. 전부 움직이면 산만해져 몰입이 깨진다.
2. **성능 안전선** — 모든 애니메이션은 `transform` / `opacity`만 사용 (GPU 가속, 저사양 폰 60fps). JS 애니메이션 루프·canvas 금지.
3. **심사 안전선** — 밝기 점멸 초당 3회 미만(광과민성), 잔혹 연출 없이 "서늘함" 유지(전연령), `prefers-reduced-motion` 사용자에겐 애니메이션 축소.

## 2. 1차 연출 상세

### A1. 암전 화면 전환
- 라우트 이동 시 검은 오버레이가 0.3~0.4초 덮었다 걷힘 (공포영화 컷 전환).
- 구현: 전역 오버레이 컴포넌트 + 라우트 변경 감지, opacity 트랜지션.

### A2. 진행할수록 어두워지는 배경
- Q1 → Q8로 갈수록 배경 텍스처 밝기 점감 (opacity 1 → 0.55) + 가장자리 비네트 점증.
- "학교 깊숙이 들어가고 있다"는 공간감. 사용자는 의식 못 해도 긴장이 쌓인다.
- 구현: 배경 img opacity를 `index/total` 기반으로 계산, transition 1초.

### A3. 내레이션 줄 단위 페이드인
- 프롤로그·선택 반응·엔딩 텍스트가 한 줄씩 0.6초 간격으로 떠오름.
- 읽는 속도를 통제해 리듬을 만드는 핵심 장치. 손글씨 폰트와 시너지.
- 탭 힌트는 마지막 줄 등장 후 표시 (다 읽기 전 넘어가는 것 방지).
- 구현: 줄 분리 후 `animation-delay: i * 0.6s` 스태거.

### A4. 귀신 갑툭튀 (클라이맥스) — story.txt 명시 연출
- 흐름: 엔딩 마지막 줄("...내가 아닌 무언가가 비쳐 있다") → 탭 → **화면 글리치 + 결과 귀신이 어두운 실루엣으로 0.3초 확 나타남** → 로딩②(또는 생략) → 결과 카드.
- **로딩 전략과 시너지**: 엔딩 화면 진입 시점에 결과 제출(`POST /histories`)을 미리 발사 → 사용자가 엔딩을 읽는 동안 응답+귀신 이미지 도착 → 갑툭튀에 **실제 내 결과 귀신** 사용, 로딩② 대부분 생략 가능.
- 가드: 점멸 3회/초 미만, 밝은 플래시 대신 실루엣+어두운 톤 ("놀람"보다 "서늘함").
- 구현: 엔딩 진입 시 submit 프리페치 / 갑툭튀 오버레이 (귀신 이미지 brightness 0.15 + scale 1.15→1 + 글리치 keyframe 0.3초).

### A5. 시작하기 버튼 심장박동 펄스
- 1.6초 주기로 미세한 scale(1→1.03) + 붉은 글로우. 시선 유도 + 두근거림 은유.

### A6. 문항 전환 크로스페이드
- 일러스트·질문·답변이 이전 것 페이드아웃 → 새 것이 아래→위 8px 슬라이드 페이드인.
- 구현: index 변경 시 key 교체 + 등장 keyframe (reaction-in 재사용).

### A7. 결과 카드 글로우 + 힌트 까딱임
- 카드 테두리에 3초 주기 붉은 글로우 펄스, 손 아이콘 상하 4px 바운스.
- 플립 인터랙션 발견율을 높이는 유도 장치.

## 3. 2차 연출 메모

- **B1 안개 드리프트**: 배경 img를 scale 1.05 상태에서 90초 주기로 translate 좌우 ±1.5%. "살아있는 배경".
- **B2 붉은 번짐**: 선택 버튼에 radial-gradient 파문 0.5초 + 나머지 요소 어둡게 → 반응 텍스트 전환과 연결.
- **B3/B4 카운트업**: requestAnimationFrame 1회성 (상시 루프 아님이라 성능 무관).
- **B5 스탬프 등장**: 이스터에그 손바닥이 페이지 로드 후 0.15초 간격으로 하나씩 scale 1.4→1 + opacity로 "찍힘".
- **B7 글리치**: 8~15초 랜덤 간격, 타이틀 text-shadow 좌우 분리 2프레임. 과하면 싸구려 느낌 — 절제 필수.
- **B8 핏자국 리플**: 탭 좌표에 소형 스플래터 0.4초 페이드. 전역 이벤트라 구현 시 스크롤 성능 확인 필요.
